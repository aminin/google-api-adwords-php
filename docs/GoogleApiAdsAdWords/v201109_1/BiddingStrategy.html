					BudgetOptimizer
					
											In budget optimizer, Google automatically places bids for the user  based on their daily/monthly budget.
										
				

								

					ConversionOptimizer
					
											Conversion optimizer bidding strategy. For more information, visit the  <a href="http://www.google.com/adwords/conversionoptimizer/index.html"  >Conversion Optimizer site</a>.
										
				

								

					ManualCPC
					
											Manual click based bidding where user pays per click. For more  information, see the article  <a href="http://adwords.google.com/support/aw/bin/answer.py?answer=159957"  >What is Enhanced CPC?</a>.
										
				

								

					ManualCPM
					
											Manual impression based bidding where user pays per thousand impressions.
										
				

								

					PercentCPA
					
											Percentage CPA based bidding where user pays a percent of conversions.
